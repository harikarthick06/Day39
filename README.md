# JavaScript Programming Constructs Practice Problems

## How to use in Antigravity / VS Code

1. Open Antigravity.
2. Create a new folder named `js-programming-constructs`.
3. Copy this project into that folder.
4. Open Terminal inside Antigravity.
5. Check Node installation:

```bash
node -v
npm -v
```

6. Run any use case like this:

```bash
node 01-sequence/01-single-digit-random.js
node 02-if-else/02-spring-season.js 21 3
node 05-for-loop/05-factorial.js 5
```

## Debugging

Use:

```bash
node inspect filename.js
```

Example:

```bash
node inspect 01-sequence/04-five-random-2digit-sum-average.js
```

Then use:
- `next` → go to next line
- `cont` → continue execution
- `.exit` → exit debugger
